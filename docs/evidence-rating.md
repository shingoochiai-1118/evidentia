# 信頼度レーティング基準 (★1〜★5)

Evidentia の星の数は、GRADE や Oxford CEBM (Centre for Evidence-Based Medicine) の
エビデンスレベルの考え方を、個人利用向けに簡略化した独自基準。「厳密な学術評価」ではなく
「日常の健康判断にどこまで頼ってよいか」の目安として使う。

## 基準

| 星 | 目安 | 典型的な根拠 |
|----|------|-------------|
| ★★★★★ | ほぼ確立した知見 | 主要な診療ガイドライン(WHO, AHA/ACC, NCCN, 日本の各学会等)が推奨し、かつ複数の質の高いメタ解析・システマティックレビューに支持されている |
| ★★★★☆ | 強い根拠だが一部留保あり | 質の高い大規模RCTが1本以上ある、または複数のメタ解析はあるが研究間の結果にややばらつきがある |
| ★★★☆☆ | 中程度の根拠 | 観察研究(コホート・症例対照)で一貫した関連が示されている、または学会が「おそらく有効」レベルで推奨、あるいは大規模RCTが1本のみで再現待ち |
| ★★☆☆☆ | 弱い根拠 | 小規模研究や専門家意見が中心、生理学的な機序の裏付けはあるが臨床的な効果検証は乏しい |
| ★☆☆☆☆ | 予備的・不確実 | 予備的データ、in vitro/動物実験のみ、または研究間で結果が一致しない |

## 特別フラグ

- **⚠️ 定説の見直し (`status: "myth_revised"`)**: かつて広く信じられていた/流行したが、その後の質の高い研究で
  効果が否定された、あるいは大幅に弱いと判明した情報。「昔は良いと言われていたが今はどうか」を明示するために使う。
  例: 健康な人へのオメガ3サプリの心血管予防効果。
- **`caution` フィールド**: 誤解しやすいポイント、副作用、対象者の限定条件などを必ず一言添える。
  「エビデンスがある」と「万人に当てはまる」は違うことを常に意識する。

## 運用ルール

1. 星の数と `evidence_level` の文言は必ずセットで記録し、どの研究・ガイドラインを根拠にしたか
   `sources` に実在するURLを最低1件添付する。記憶だけに基づく不確かな数値・研究名は書かない。
2. 一次情報(ガイドライン本文、RCT論文、システマティックレビュー)を優先し、X/Facebook/ニュース記事などの
   二次情報しか見つからない場合は、その旨を `caution` に明記し、星は1段階以上下げて保守的に評価する。
3. エビデンスは更新される。半年〜1年に一度は `last_reviewed` を見直すことが望ましい
   (自動化は [[roadmap]] の Phase 3 で検討)。

## 優先する情報源(ドメイン)

検索は汎用のWeb検索を使うが、以下のドメインを優先的に信頼する。PubMed本体やGoogle Scholarへの
直接アクセス手段は無いが、これらのドメインで見つかる情報はPubMed/PMC収載論文や主要ガイドラインと
実質的に重なる。

**英語・国際(一次エビデンスの主軸)**
- `pubmed.ncbi.nlm.nih.gov` / `ncbi.nlm.nih.gov` (PubMed Central)
- `cochranelibrary.com` / `cochrane.org`
- `nejm.org`, `thelancet.com`, `jamanetwork.com`, `bmj.com`
- `who.int`, `cdc.gov`, `nih.gov`(NHLBI/NIAなど下部組織含む)
- `uspreventiveservicestaskforce.org`, `fda.gov`

**日本語(国内ガイドライン・検診年齢・薬事承認状況の確認用)**
- `mhlw.go.jp`(厚生労働省)
- `ncc.go.jp` / `ganjoho.jp`(国立がん研究センター・がん情報サービス)
- `j-circ.or.jp`(日本循環器学会)
- `jhsnet.net`(日本頭痛学会)
- `joa.or.jp`(日本整形外科学会)
- `jasso.or.jp`(日本肥満学会)

## 検索言語の方針

- **英語**を主軸にする。RCT・メタ解析・国際的な診療ガイドラインは国籍を問わずほぼ英語で発表・要約されるため、
  質の高い一次エビデンスを探す際は英語検索を基本とする。
- **日本語**は、日本在住者にとって実務上重要な差分(検診対象年齢、薬剤の国内承認状況、日本人特有の
  リスク因子・基準値など)を確認するために併用する。特に医薬品・検診・肥満基準(BMI等)は国によって
  推奨が異なることがあるため、日本語での確認を省略しない。
- 英語・日本語以外の言語(独仏中韓等)は追加しない。臨床上重要なエビデンスは最終的にほぼ英語で
  公表・要約されるため、言語を増やしてもレビューの手間が増えるだけで得られるものが少ないと判断する。

## 情報の鮮度について

星の数は「エビデンスの質」を表すものであり、「新しさ」とは別軸である。10年以上前の研究・ガイドライン
であっても、その後の研究で覆っていなければ星を下げる理由にはならない(例: アルコールの発がん性分類は
1987年、PREDIMED試験は2013年、いずれも現役のエントリーとして採用済み)。「最新の話題を扱う」ことは
別途 [[workflow]] の「医療の専門知識」(新薬・既存薬の新しい使い方)という項目が担当しており、
鮮度と信頼度を混同しない。

## エビデンスが覆った場合の扱い(削除ではなく更新)

過去に追加したエントリーの結論が、その後の研究で覆った・弱まったと判明した場合、エントリーを削除は
せず、次のように更新する。
1. `status` を `"myth_revised"` に変更する
2. `caution` に何が・いつ・どう変わったかを追記する(地中海食エントリーの「2018年撤回・再発表」の
   ような記録と同じ考え方)
3. `last_reviewed` を更新日に変更する

「昔は良いとされていたが今は違う」という変遷の記録自体に価値がある、というのがこのサイトの方針。
ユーザーが明示的に削除を指示した場合のみ、エントリーそのものを取り除く([[workflow]]の
「エントリーの削除・修正」参照)。
