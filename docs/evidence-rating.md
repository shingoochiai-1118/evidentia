# 信頼度レーティング基準 (★1〜★5)

Evidentia の星の数は、GRADE や Oxford CEBM (Centre for Evidence-Based Medicine) の
エビデンスレベルの考え方を、個人利用向けに簡略化した独自基準。「厳密な学術評価」ではなく
「日常の健康判断にどこまで頼ってよいか」の目安として使う。

## 基準

| 星 | 目安 | 典型的な根拠 |
|----|------|-------------|
| ★★★★★ | ほぼ確立した知見 | 主要な診療ガイドライン(WHO, AHA/ACC, NCCN, 日本の各学会等)が推奨し、かつ複数の質の高いメタ解析・システマティックレビューに支持されている |
| ★★★★☆ | 強い根拠だが一部留保あり | 質の高い大規模RCTが1本以上ある、または複数のメタ解析はあるが研究間の結果にややばらつきがある |
| ★★★☆☆ | 中程度の根拠 | 観察研究(コホート・症例対照)で一貫した関連が示されている、または学会が「おそらく有効」レベルで推奨、あるいは大規模RCTが1本のみで再現待ち |
| ★★☆☆☆ | 弱い根拠 | 小規模研究や専門家意見が中心、生理学的な機序の裏付けはあるが臨床的な効果検証は乏しい |
| ★☆☆☆☆ | 予備的・不確実 | 予備的データ、in vitro/動物実験のみ、または研究間で結果が一致しない |

## 特別フラグ

- **⚠️ 定説の見直し (`status: "myth_revised"`)**: かつて広く信じられていた/流行したが、その後の質の高い研究で
  効果が否定された、あるいは大幅に弱いと判明した情報。「昔は良いと言われていたが今はどうか」を明示するために使う。
  例: 健康な人へのオメガ3サプリの心血管予防効果。
- **`caution` フィールド**: 誤解しやすいポイント、副作用、対象者の限定条件などを必ず一言添える。
  「エビデンスがある」と「万人に当てはまる」は違うことを常に意識する。

## 運用ルール

1. 星の数と `evidence_level` の文言は必ずセットで記録し、どの研究・ガイドラインを根拠にしたか
   `sources` に実在するURLを最低1件添付する。記憶だけに基づく不確かな数値・研究名は書かない。
2. 一次情報(ガイドライン本文、RCT論文、システマティックレビュー)を優先し、X/Facebook/ニュース記事などの
   二次情報しか見つからない場合は、その旨を `caution` に明記し、星は1段階以上下げて保守的に評価する。
3. エビデンスは更新される。半年〜1年に一度は `last_reviewed` を見直すことが望ましい
   (自動化は [[roadmap]] の Phase 3 で検討)。
